# Product Stack

An overview of the tools and services we use to build and run Stunt Double.

![Alt](https://repobeats.axiom.co/api/embed/a17878ddc109157f729b6faad269b9514ee84c80.svg "Repobeats analytics image")

## Daily Drivers

- **[Claude Code](https://claude.ai/code)** + **[Cursor](https://cursor.sh/)**: AI-first development — Claude Code for agentic workflows, Cursor for in-editor prompting
- **[Arc](https://arc.net/)**: Modern browser with Spaces and Boosts
- **[Linear](https://linear.app/)**: Tasks, planning, and prioritisation
- **[Slack](https://slack.com/)**: Team communication
- **[1Password](https://1password.com/)**: Passwords, secrets, and certificates
- **[Raycast](https://www.raycast.com/)**: Launcher and productivity workflows
- **[Homebrew](https://brew.sh/)**: macOS package manager

## Engineering

- **[TypeScript](https://www.typescriptlang.org/)**: One language across the full stack
- **[Next.js 16](https://nextjs.org/)**: App Router, Server Components, React 19
- **[NX](https://nx.dev/)**: Monorepo workspace management with incremental builds and caching
- **[pnpm](https://pnpm.io/)**: Fast, disk-efficient package manager
- **[Vercel](https://vercel.com/)**: Hosting, previews, analytics, feature flags, and toolbar
- **[Supabase](https://supabase.com/)**: Postgres database, auth, storage, and realtime
- **[Trigger.dev v4](https://trigger.dev/)**: Background jobs, AI workflows, and orchestration
- **[Cloudflare Workers](https://workers.cloudflare.com/)**: Edge compute (email processing)
- **[Tauri](https://tauri.app/)**: Native macOS desktop app
- **[Zod](https://zod.dev/)**: Schema validation
- **[GitHub](https://github.com/)**: Source control and CI/CD via Actions

## Design

- **[Figma](https://www.figma.com/)**: Design, slides, prototyping, and design system
- **[shadcn/ui](https://ui.shadcn.com/)**: Component library built on Radix UI
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Utility-first styling
- **[Lucide](https://lucide.dev/)**: Icon library
- **[Motion](https://motion.dev/)**: Animation library
- **[Klim](https://klim.co.nz/)**: Fonts
- **Rotato**: Product mockups and video

## AI and Machine Learning

- **[Anthropic Claude](https://www.anthropic.com/)**: Primary LLM — via direct SDK, Vertex AI, and Agent SDK
- **[Vercel AI SDK v6](https://sdk.vercel.ai/)**: Unified interface across providers
- **[OpenAI](https://openai.com/)**: Embeddings
- **[Google Vertex AI](https://cloud.google.com/vertex-ai)**: Claude models via Google Cloud
- **[MCP (Model Context Protocol)](https://modelcontextprotocol.io/)**: AI tool integration
- **[Browserbase](https://www.browserbase.com/)** + **[Stagehand](https://stagehand.dev/)**: Browser automation for AI agents
- **[Tavily](https://tavily.com/)**: Web search for AI
- **[Claude Desktop](https://claude.ai/)**: Local AI assistant

## Payments and Email

- **[Stripe](https://stripe.com/)**: Payment processing
- **[Resend](https://resend.com/)**: Transactional email
- **[React Email](https://react.email/)**: Email templates

## Observability

- **[Vercel Analytics](https://vercel.com/analytics)**: Web analytics
- **[Vercel Speed Insights](https://vercel.com/docs/speed-insights)**: Core Web Vitals
- **[Vercel Flags](https://vercel.com/docs/workflow-collaboration/feature-flags)**: Feature flags
- **[rrweb](https://www.rrweb.io/)**: Session recording and replay
- **[Pino](https://getpino.io/)**: Structured logging

## Security and Networking

- **[Cloudflare Zero Trust](https://www.cloudflare.com/zero-trust/)**: Domain access, private VPN (free up to 50 users)
- **[Supabase Auth](https://supabase.com/auth)**: Authentication with Google OAuth and magic links

## Operations

- **BusyCal**: Calendar
- **Reclaim.ai**: Smart scheduling
